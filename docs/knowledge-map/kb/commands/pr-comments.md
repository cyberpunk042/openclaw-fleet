# /pr-comments

**Type:** Claude Code Slash Command
**Category:** Built-in
**Available to:** FLEET-OPS, ENG

## Purpose

Fetch GitHub PR review comments for the current branch.

## When to Use

Read review feedback. Process rejection comments. Understand what reviewers flagged.

## Methodology Stages

review

## Relationships

fleet_approve tool, fleet-review skill (read feedback), PR workflow
