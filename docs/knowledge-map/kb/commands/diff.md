# /diff

**Type:** Claude Code Slash Command
**Category:** Built-in
**Available to:** ENG, QA, FLEET-OPS

## Purpose

Interactive diff viewer showing uncommitted and per-turn changes.

## When to Use

Review own changes before committing. Fleet-ops review Step 2. Compare turn-by-turn changes.

## Methodology Stages

work (before commit), review

## Relationships

fleet_commit (review changes first), fleet-review skill (Step 2 read the diff), PR review workflow
