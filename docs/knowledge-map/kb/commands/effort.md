# /effort

**Type:** Claude Code Slash Command
**Category:** Built-in
**Available to:** ALL

## Purpose

Set extended thinking effort level: low, medium, high, max (opus only), auto.

## When to Use

high for architecture decisions. max for critical security analysis. low for simple updates.

## Methodology Stages

any

## Relationships

model_selection.py, CLAUDE_CODE_EFFORT_LEVEL env var, skill frontmatter (can override)
