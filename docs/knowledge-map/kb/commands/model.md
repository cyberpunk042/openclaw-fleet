# /model

**Type:** Claude Code Slash Command
**Category:** Built-in
**Available to:** ALL

## Purpose

Select or change AI model. Left/right arrows adjust effort slider.

## When to Use

Switch to opus for complex reasoning. Switch to sonnet for routine work. Match model to task complexity.

## Methodology Stages

any

## Relationships

model_selection.py, effort levels (low/medium/high/max), injection profiles (model drives profile selection)
